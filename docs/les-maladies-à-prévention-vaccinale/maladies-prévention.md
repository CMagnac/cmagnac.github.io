# Les maladies à prévention vaccinale

Les vaccins protègent contre de nombreuses maladies. Les informations présentes dans cette partie ne sont pas exhaustives. Ce document n’aborde pas les mesures préventives autres que la vaccination comme le lavage des mains et les mesures hygiéno-diététiques.

# Personal Portfolio & Blog

A personal website built with [Astro](https://astro.build/) and the [Astroplate](https://github.com/zeon-studio/astroplate) starter, designed to showcase my work through **portfolio projects and blog posts**.

This site also highlights how I use **AI tools** responsibly in my workflow to design, build, and iterate on projects focused on **healthcare professionals**.

## Tech stack

* [Astro](https://astro.build/)
* [Astroplate](https://github.com/zeon-studio/astroplate)
* Tailwind CSS
* Markdown / MDX content
* GitHub Pages for deployment
* GitHub Actions for automated builds

## AI-assisted development

I use AI tools as part of my development workflow. AI is a **support tool**, not a replacement for engineering judgment.

**Every feature, decision, and final implementation is reviewed, tested, and adapted by me**.

## Healthcare-focused projects

Some of the projects showcased on this website may focus on **healthcare professionals**. These projects are intended for **software demonstration** and **professional portfolio** purposes.

## Deployment

The website is intended to be deployed on GitHub Pages from the `main` branch with GitHub Actions.
